Getting Started

In case you ever wondered, HTML (HyperText Markup Language) is used to create all the content you see on any given website.

What you'll learn

In this course you'll learn a lot of different things like "yada yada yada"

Basic HTML Structure

Basically, HTML uses three pieces:

  • An optional section that contains metadata about the page called the head
  • A section that contains the content of your webpage, called the body
  • An optional section that contains contact data or other site links called the foot

Code Example

        
          <html>
              <head>
              </head>
              <body>
              </body>
              <footer>
              </footer>
          </html>
